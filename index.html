Master Equation
The one rule that makes everything work
f₁ = f₂ = d_sep
Mirror Geometry
Focal length from diameter & depth
f = D² / (16d)
Depth from focal length & diameter
d = D² / (16f)
Diameter from focal length & depth
D = 4√(f·d)
Radius of curvature at vertex
R = 2f
Canonical Relationships (rims touching)
Focal length from diameter
f = D / (2√2) ≈ 0.3536 × D
Dish depth from diameter
d = D / (4√2) ≈ 0.1768 × D
Vertex separation
d_sep = f = 2d
f-ratio (fixed for all sizes)
N = f/D = 1/(2√2) ≈ 0.354
Image Position & Magnification
Image height (given object height)
h_img = f·h_obj / (f − h_obj)
Object height (given desired image height)
h_obj = f·h_img / (f + h_img)
Magnification
m = f / (f − h_obj)
Lateral inversion
x_img = −x_obj × m left↔right swap, NOT upside down
Viewing Cone (angles from horizontal)
Minimum angle (inner limit)
θ_min = arctan(A/2 / d_sep) = arctan( (A/D)·√2 )
Maximum angle (outer limit) — FIXED
θ_max = arctan(√2) = 54.74° always
Cone width
Δθ = θ_max − θ_min
Optimal viewing
θ_opt = (θ_min + θ_max) / 2
Peep risk ratio
A/D < 0.25 for low peep risk
Aperture/diameter ratio
A/D = tan(θ_min) / tan(θ_max)
Ray–Parabola Intersection
Ray from (ox, oy) with direction (dx, dy), parabola vertex (vx, vy), sign s = ±1
u = ox − vx,  v = oy − vy
A = dx²
B = 2u·dx − 4f·s·dy
C = u² − 4f·s·v
Solve At² + Bt + C = 0, take smallest positive t If dx = 0 (vertical ray): t = −C/B
Reflection
Surface normal at (x, y) on parabola
n_x = −(x − vx) / (2f)
n_y = s
then normalize: n̂ = n/|n|
Reflected ray direction
r = d − 2(d·n̂)n̂ d = incident direction, n̂ = unit normal
Brightness
Net image brightness
I_img = I_obj × R₁ × R₂ × η R₁,R₂ ≈ 0.90 (aluminum). η ≈ 0.5–0.6 (geometric efficiency)
Secondary Images (d_sep > f)
Mirror separations where secondary images appear
4 reflections:  d_sep/f ≈ 1.39
6 reflections:  d_sep/f ≈ 1.56
8 reflections:  d_sep/f ≈ 1.66
n → ∞:        d_sep/f → 2.0
Hard Constraints
Object must be below focal point
h_obj < f
Aperture smaller than diameter
A < D
Dishes must not intersect
d < d_sep / 2
Image must form (rays converge)
d_sep < 2f
